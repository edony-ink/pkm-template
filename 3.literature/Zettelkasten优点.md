# Zettelkasten优点
Created: 2021-09-14 16:57

总结下来Zettelkasten有如下优点：
1. 利于更好学习和理解知识（建立连接）
2. 简单高效（note记录类似one-liner负担较轻，通过构建graph来解决结构化问题）
3. 减轻大脑负担（不需要记住大篇幅的东西，通过连接构建知识体系）
4. 增加创造性（每个单点之间联系会产生意想不到的东西，e.g. 钢琴 + 写作 = 打字机）
